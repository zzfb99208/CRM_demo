# CRM Demo 系统 — 启动指南

> 从零开始启动 CRM Demo 系统的后端和前端，适用于 Windows 系统。

---

## 前置条件（已完成，无需重复操作）

| 组件 | 状态 | 说明 |
|------|------|------|
| JDK 24 | ✅ | `C:\Program Files\Java\jdk-24` |
| Maven 3.9.9 | ✅ | `C:\Users\张富渤\maven\apache-maven-3.9.9` |
| Node.js 22 | ✅ | 系统 PATH 中 |
| MySQL 8.0 | ✅ | 服务名 MYSQL80，用户名 `root`，密码 `root` |
| 数据库 | ✅ | `crm_demo`，含 10 张表 + 种子数据 |

## 一、启动后端（Spring Boot）

### Step 1：打开命令行

按 `Win + R`，输入 `cmd`，回车。

### Step 2：进入后端目录 + 设置 Maven 路径

在命令行中依次输入：

```bash
cd /d D:\Desktop\CRM_demo_ws\crm-demo-backend

set PATH=C:\Users\张富渤\maven\apache-maven-3.9.9\bin;%PATH%
```

### Step 3：启动后端

```bash
mvn spring-boot:run
```

首次启动会下载依赖，需要 1~2 分钟。看到以下信息表示启动成功：

```
Started CrmDemoApplication in X.XXX seconds
```

后端运行在 **`http://localhost:8080`**。

验证：浏览器打开 `http://localhost:8080/api/customers/search?keyword=AMD`，应返回 JSON 数据。

---

## 二、启动前端（Vue 3 + Vite）

### Step 4：再打开一个命令行窗口

**不要关闭后端窗口**。按 `Win + R`，输入 `cmd`，回车，打开第二个命令行窗口。

### Step 5：进入前端目录

```bash
cd /d D:\Desktop\CRM_demo_ws\crm-demo-web
```

### Step 6：启动前端开发服务器

```bash
npm run dev
```

看到以下信息表示启动成功：

```
  VITE v5.x.x  ready in xxx ms

  ➜  Local:   http://localhost:5173/
```

前端运行在 **`http://localhost:5173`**。

## 三、使用系统

浏览器打开 `http://localhost:5173`，左侧菜单导航：

| 菜单 | 功能 |
|------|------|
| 客户管理 | 搜索客户、查看详情、导入 PO |
| 产品总表 | 查看全部产品 |
| 采购订单 | PO 列表、导入、查看 |
| 形式发票 | PI 列表、查看 Invoice、导出、重新导入修改后的 PI |
| 装箱单 | PL 列表、查看 Packing List、导出 |

### 完整操作流程

```
Step 1: 客户管理 → 搜索 "AMD" → 点击"查看"
Step 2: 客户详情 → 点击"导入 PO Excel"
Step 3: 上传 Excel 文件 → 系统解析 → 确认
        （用 D:\Desktop\CRM_demo_ws\相关示例文件\清洁版 CIP_IT-AMD-20260130 pending orders.xlsx）
Step 4: PO 详情 → 点击"生成 PI"
Step 5: PI 详情 → 查看 Invoice → 可导出 PI / 导入修改后的 PI
Step 6: PI 详情 → 点击"生成 Packing List"
Step 7: PL 详情 → 查看 Packing List → 导出 PL
```

## 四、停止系统

- **后端**：在运行后端的命令行窗口中按 `Ctrl + C`，输入 `Y` 确认
- **前端**：在运行前端的命令行窗口中按 `Ctrl + C`

## 五、常见问题

| 问题 | 原因 | 解决 |
|------|------|------|
| `mvn` 命令找不到 | PATH 未设置 | 执行 `set PATH=C:\Users\张富渤\maven\apache-maven-3.9.9\bin;%PATH%` |
| 端口 8080 被占用 | 上次后端未正常关闭 | 任务管理器结束 Java 进程，或 `netstat -ano | findstr 8080` 查 PID 后 `taskkill /F /PID xxx` |
| 端口 5173 被占用 | 上次前端未正常关闭 | 关闭对应命令行窗口 |
| 数据库连接失败 | MySQL 服务未启动 | 管理员身份运行 `net start MYSQL80` |
| MySQL 密码错误 | 密码被重置 | 管理员身份运行 `D:\Desktop\CRM_demo_ws\reset_mysql_password.bat` |
| 前端页面空白 | 后端未启动 | 确保后端先启动，再刷新前端页面 |
